HTML/CSS




Stap 1: Map maken

Om te beginnen voeg je een nieuw mapje toe in je Finder.
Het mapje kan je benoemen naar je eerste les. Bijvoorbeeld: les-01


Stap 2: Visual Studio Code

Je opent Visual Studio Code. Er zijn ook andere programma's,
maar deze is erg fijn om mee te werken. Nu kan je het nieuwe mapje
openen door op "open folder" te klikken. De map staat erin.

Stap 3: Bestand maken

Als je in het mapje staat, kan je klikken op het eerste icoontje
met een plusje naast de naam van je map. Wanneer je hierop klikt
is er een nieuw bestand aangemaakt. Dat bestand benoem je met de naam "index.html".
Hierin ga je aan de slag met html.

Stap 4: HTML schrijven

Om html te schrijven heb je allemaal basis tags nodig.
Deze maken het mogelijk om iets te zien op jouw site. Hiernaast zie je een voorbeeld.

Stap 5: CSS bestand maken

Je wilt er nu natuurlijk ook vormgeving aan geven.
Je klikt op hetzelfde icoontje als bij stap 3 en benoemt het bestand "style.css".

Stap 6: CSS linken

Om ervoor te zorgen dat deze gelinkt is aan je html bestand, moet je dat in html melden.
Zie afbeelding hiernaast. Als je dat niet doet zijn je vormgevings aanpassingen ook niet te zien op je site.
